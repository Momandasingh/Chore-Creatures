# 🐾 Chore Creatures

A fun, competitive chore tracker for kids — powered by virtual pets that get happier as tasks get done.

## What it is

Chore Creatures is a single-file web app that turns daily routines into a friendly competition between siblings. Each child has a virtual pet that reacts to their progress. The more tasks they complete, the happier their pet becomes. Kids can see each other's progress side by side, which turns brushing teeth and making the bed into something they actually want to do.

**Live app:** [chorecreatures.netlify.app](https://chorecreatures.netlify.app)

## Features

- **Side-by-side pet cards** with a live ⚔️ VS scoreboard
- **Happiness bar** that fills as tasks are completed, with sparkles and confetti when a child finishes everything
- **Morning, Afternoon, and Evening task sections** so routines are organized by time of day
- **Customizable pets** — 18 animals to choose from, with a custom name for each
- **8 background themes** — Purple, Ocean, Forest, Sunset, Candy, Night, Berry, and Mint
- **2 to 4 kids** supported, with the layout adjusting automatically
- **Drag-and-drop task reordering** plus ▲▼ buttons for touch devices
- **Add, remove, and reassign tasks** by time period from Parent Settings
- **Reset for New Day** button to clear all checkmarks each morning
- **Coin system** — Earn coins by completing tasks, with a daily +50 bonus for finishing everything
- **Pet shop with 24 items** — Hats, Accessories, Toys, and Treats to customize and dress up your pet
- **No account needed** — all data is stored locally in the browser, so each family has their own private copy

## How to use

Just open `task-pets.html` in any browser — Chrome, Safari, or Edge on a phone, tablet, or computer. No installation, no login, no internet connection required after the first load.

To set up for your family, open **Parent Settings** at the bottom of the page to:
- Enter each child's name
- Customize the task list
- Pick a background theme

Tap the **✏️ Edit** button on any pet card to change the animal and give the pet a name.

## Sharing

Because all data is stored in the browser, anyone you share the link with will see a blank app ready for their own family to set up. No one can see your data and you can't see theirs.

## Tech

A single self-contained HTML file with vanilla JavaScript and CSS. No frameworks, no dependencies, no build step. Works offline once loaded.
